## 核心改进

### 1. 跌日评分修正
- 启用WR超买惩罚（`use_wr=1`, `wr_lo_b=-15`）
- 降低macd权重（`macd_w 3.0→1.5`）、cl权重（`cl_w 0.4→0.2`）
- 提高p权重（`p_w 0.5→1.8`）

### 2. 横盘评分修正
- 移除WR超买错误加分（原`w<30:bs+=8`→`w>80:bs+=5`仅保留超卖加分）
- `d>0.3 and w<30: bs+=3` 改为 `d>0.3 and w>50: bs+=3`

### 3. 7天动量衰减检查（通用，所有行情适用）
- **规则1**：p是7天最高+7天均线很低 →分级扣分（-5~-15）
- **规则2**：V型反弹（前2天深跌+今天暴涨+均线弱）→ -8
- **规则3**：近期大涨后连续下跌 → -10
- **规则5**：5天前峰值已过+近期偏弱 → -8
- **规则6**：极端超买(WR<10)+7天弱+今天最高 → -8
- **规则4**：严格递减4天 → -10
