# 大道至简 · 基本准则

> 所有策略生成、策略进化、策略探索，必须基于本基本准则进行高胜率回测研究。

---

## 一、整体核心规则

1. **不同行情**，使用专属独立选股策略。
2. **每套策略配合同套评分机制**——4行情×4组（策略+评分）独立配对。
3. 策略与评分**版本对应**，相互相成，版本一体迭代。
4. 每套策略拥有**独立的分级放宽规则**（L→L5各自定义放宽参数）。
5. **目标**：冠军（评分第1名）次日最高涨幅 ≥2.5%，持续提高胜率。

## 二、五级筛选规则（优先级从高到低：L → L1 → L2 → L3 → L4 → L5）

1. **从最高级L级开始**选股，当前级选出 **＞8只** → 立即终止，锁定该级票池。
2. 当前级 **≤8只** → 自动降级到下一级（放宽条件）。
3. **5级全部过完仍≤8只** → 当日弃权（不交易）。
4. **各行情策略独立定义**自己的L→L5放宽参数（涨/量比/换手/CL/市值逐级放宽幅度不同）。
5. 放宽方式：沿用目前的参数逐级放宽模式（每级降门槛、升上限）。

## 三、固定基础选股条件

1. 标的：沪深主板A股（600/601/603/605/000/001/002）
2. 剔除：ST、\*ST、退市股、次新股（上市满1年以上）
3. 当日涨幅：小于8%
4. 交易预期：次日冲高涨幅 ≥2.5%，持续提高胜率

## 四、完整执行流程

1. **判断当日行情** → 匹配对应专属策略
2. **从L级开始逐级筛选** → 超8只即停，≤8只则降级
3. **锁定票池**后，**用该策略的配套评分公式**打分（4行情×4独立评分）
4. **取评分第1名**个股
5. **核算该形态历史胜率**，输出最终选股结果（含次日最高涨幅%）
