# 大道至简 · 公共评分基准 V260528

> **本基准为所有策略优化的参照对比标准。**
> 任何新版本的评分优化，必须回测对比本基准的冠军胜率。
>
> **评估优先级**：近30天胜率 > 80天/全量胜率。
> 若30天胜率低于80天，说明策略仍在退化，需继续优化。

---

## 一、评分体系原则

1. **4行情×4独立评分** — 各行情策略使用自己的子策略文件定义的评分权重。
2. **评分文件与策略配对** — 策略与评分版本对应，一体迭代。
3. **评分文件位置**：`大道至简_子策略01~04.py`，各文件内`SCORE`字典定义该行情评分权重。
4. **通用加分项（可选）** ：V260528验证了三项通用规则可叠加上去（透支惩罚、MACD加分），按行情实测决定是否采用。

## 二、基础评分公式

```
score = p × p_w + CL × cl_w + 价格分 × 0.3 + MACD分 × macd_w

各加分项（权重从子策略文件SCORE加载）:
  + (站上MA5 ? ma5_b : 0)
  + (量比1.0~1.5 ? vr_b × 1.5 : 0)
  + (换手5~7% ? hs_b × 2 : 0)
  + (WR < 25 ? wr_b : 0)
  + (J > K > D ? j_b : 0)
  + (20 ≤ J ≤ 40 ? j_low_b : 0)
  + (KDJ金叉 ? kdj_b : 0)

V260528通用加分（各行情可选，需实测决定）:
  if p > 5% and CL > 80%: score -= 8    # 透支重罚
  if dif > 0.5:           score += 3    # MACD势头加分
  if MACD金叉:            score += 3    # 金叉加分
```

## 三、辅助函数

### 价格分
```
价格分 = min(10, max(1, 11 - 买入价 / 10))
```

### MACD分
```
if MACD金叉 and dif > 0.5:  return 10
if MACD金叉 and dif > 0.2:  return 8
if MACD金叉:                return 6
if dif > 0.5:               return 4
if dif > 0:                 return 2
else:                       return 0
```
